# 1. Установка и запуск

## 1.1. Создание и активация виртуального окружения

**Linux / macOS**
```bash
python -m venv .venv
source .venvenv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```

**Windows (PowerShell)**
```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install --upgrade pip
pip install -r requirements.txt
```
